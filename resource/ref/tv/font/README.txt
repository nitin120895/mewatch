Go to https://icomoon.io/app and upload there the selection.json.
Click on "Yes", that you want to load all settings from selection.json.
Add your new icons.
And generate new fonts and update the selection.json.


Each font pack you download from the IcoMoon app comes with a selection.json file which
you can import back to the app in order to modify or update your font and the class names
associated with its glyphs.
